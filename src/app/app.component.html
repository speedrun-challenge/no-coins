Challenge: Get the highest combined "No Coins" times in Subway surfers and Temple Run 2 until September 4th!

Current prize pool: $100
(You want to add to the prize pool? See contact information below)

Participate

Participation is free. Once you have registered, you can submit as many runs on the leaderboard as you want until the challenge ends. Your best (longest) runs will be considered for the bounty.

How to participate:
  1. Submit your speedrun.com name here: Registration
  2. Submit a run to both Subway surfers "No Coins" (with "No hoverboard/keys") and Temple Run 2 "No Coins" on the speedrun.com leaderboards.
  3. The runner(s) with the combined longest runs (see example below) will win the bounty.

Example: A Subway Surfers "No Coins" run of 1:35.200 and a Temple Run 2 "No Coins" run of 1:10.100 will result in a combined time of 2:45.300.

Rules

  • Only runs submitted before 4th of September, 23:59 BRT (= September 5th 02:59 UTC) will be considered.
  • Runs must be submitted to speedrun.com and therefore follow the respective rules of the game categories. Only runners with a run in both games are eligible for the bounty. The runs must be in the respective "No coins" category, and for Subway Surfers, the run must be of type "No hoverboard/keys".
  • Distribution of prize pool: Below $300: 1 winner (100%), between $300-500: 2 winners (70%-30%), Above $500: 3 winners (50%-30%-20%)
  • Winners will be announced publicly and also be contacted directly via the speedrun.com message system for security measures. Be aware that it may take 2-3 weeks to evaluate the winner, depending on the number of participants.
  • Prizes will be paid out via PayPal or wire transfer.
